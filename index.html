Features

Free And Open Source

Don’t worry about licensing fees because there aren’t.

The openness of this project also ensures that anyone can participate in making this project better.

Customisable

From background images, fonts, object locations and measurements, you can customize the design of the certificate as you wish*.

Web-Based Interface

Whether you're using your phone, tablet or PC, it's just works!**

Access the system wherever you are.

*There is a difference in the actual size of the object on the certificate compared to that on the layout editing page.

**For general purpose use, this system works with any device. For the event certificate layout editing page, bigger screen devices is necessary.

Built For Community

This is an open source project. It will always be free to be used by anyone.

Whether you are an individual, university and schools, companies or anyone who hosted an event and wanted to hand out e-certificate to participants. This project is for you.

Got any ideas and improvements to be added to this project?

Feel free to open an issue to the GitHub repository.
